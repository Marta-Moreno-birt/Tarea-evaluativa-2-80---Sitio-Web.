Paquete Basic

Este paquete incluye el diseño de logotipo básico y el diseño y creación de una web con páginas estáticas de hasta 5 páginas.

El logotipo será entregado de manera online en formato png. Además, recibirás diferentes variantes del mismo, así como las aplicaciones de este: Diseño de tarjetas de visita, diseño de banner para web y diseño de marca de agua para documentos corporativos.

En cuanto a la web. Se trata de una web básica con hasta 5 páginas estáticas como máximo. Este diseño está pensado para webs corporativas como por ejemplo un despacho de abogados, una asesoría jurídica...etc Incluirá en una de sus páginas un formulario para que el cliente pueda ponerse en contacto con la empresa en cuestión.
